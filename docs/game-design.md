# Math Odyssey（数学奥德赛）— Game Design Document

## Context
美国高中数学学习游戏。玩家修复废弃太空站，数学是解决问题的工具。融合任务驱动 + 解谜 + 建造 + 经营。React + Vite，赛博朋克霓虹风。

## 核心游戏设计

### 故事背景
飞船坠落在废弃太空站"Odyssey Station"。太空站的 5 个核心系统全部瘫痪，玩家需要用数学修复每个系统才能重新启航。每修好一个系统，太空站可见地变化（灯亮、区域解锁、新功能上线）。

### 核心循环
**任务** → **互动解谜**（数学是工具）→ **顿悟时刻** → **系统修复**（可见奖励）→ **站点升级**

### 乐趣来源
- 数学不是"要学的东西"，是"修好太空站的超能力"
- 每个谜题有"啊哈！"时刻——你自己发现规律，不是被告知公式
- 太空站从黑暗废墟 → 灯火通明的基地，成就感可见
- 没有错误惩罚：所有操作都有反馈，引导你理解

### Algebra 1 → 太空站系统映射

| 太空站系统 | 数学主题 | 3个修复任务 |
|-----------|---------|-----------|
| ⚡ Power Core（能源核心）| 数与表达式 | 校准反应堆坐标(数轴) → 解码启动序列(运算顺序) → 优化配电(运算性质) |
| 🧭 Navigation（导航系统）| 一元一次方程 | 平衡推进器(方程概念) → 单次轨道修正(一步方程) → 复杂航线修正(多步方程) |
| 📡 Comms Array（通信阵列）| 线性函数 | 信号模式识别(函数) → 天线对准(斜率) → 频率调谐(y=mx+b) |
| 🛡️ Shield Gen（护盾发生器）| 方程组 | 护盾重叠区(图解法) → 频率替换(代入法) → 谐波消除(消元法) |
| 🚀 Launch Seq（发射序列）| 二次函数 | 发射弧线(抛物线) → 结构强度(因式分解) → 逃逸速度(求根公式) |

## Mission 详细设计

### Mission 1: 校准反应堆坐标（实数轴）— ✅ 已完成
**剧情：** 反应堆的燃料棒需要插入精确坐标位置才能激活。控制面板显示一条校准轴。
**玩法：**
- 一条可缩放/平移的数轴，上面有若干空的燃料槽位（发光虚框）
- 任务指令："将燃料棒放到坐标 -3 处"、"放到 π 附近"、"放到 1/2 和 1 之间"
- 拖拽燃料棒到正确位置，位置对了 → 燃料棒亮起，反应堆功率上升
- 渐进任务：整数 → 分数 → 无理数，每次成功发现新概念
**顿悟时刻：**
- 放好 π 时："原来有些数永远写不完——这就是无理数！"
- 放好对称位置时："正数和负数是镜像的！"
**完成奖励：** 反应堆启动，太空站底部灯光亮起，解锁下一任务

### Mission 2: 解码启动序列（运算顺序 PEMDAS）
**剧情：** 反应堆启动了但启动序列被打乱，控制台显示一组运算指令，需要按正确顺序执行。
**玩法：**
- 屏幕显示表达式如 `2 + 3 × 4`，结果面板实时显示计算值
- 玩家点击/拖拽选择"先算哪一步"，每一步高亮并执行
- 如果按错误顺序（先算加法），反应堆输出异常值 → 功率指示灯闪红（不是惩罚，是反馈）
- 发现不同顺序给出不同结果后，PEMDAS 规则自然浮现
- 后续关卡：加入括号，让玩家主动用括号改变执行顺序
**顿悟时刻：**
- "同样的数字，顺序不同结果就不同！"
- "括号可以强制改变计算优先级！"
**完成奖励：** 启动序列正确执行，能源核心全功率运行

### Mission 3: 优化配电网络（运算性质）
**剧情：** 能源核心运行了，但配电网络效率低下。需要重新排列电路来优化。
**玩法：**
- **交换律谜题：** 两组电路 `A→B` 和 `B→A`，拖拽交换，电表显示功率不变 → "顺序不影响结果"
- **结合律谜题：** 三段串联电路 `(A+B)+C`，拖拽括号重新分组为 `A+(B+C)`，功率不变 → "分组不影响结果"
- **分配律谜题：** 一个大电池供电给两个并联设备，可视化为面积矩形：`3×(4+2)` = `3×4 + 3×2`。拖拽分割线拆分/合并
- 每个性质理解后，配电效率提升一档
**顿悟时刻：**
- 分配律面积模型："原来乘法分配就是把一个大矩形拆成两个小矩形！"
**完成奖励：** 配电网络优化完成，太空站全区照明亮起
